# Tool Spy Idea v1.0.0
### by ChThanh

Spy products from Etsy, Shopify + Download images + Dropbox links + Clean Amazon titles.

---

## 🚀 Cách chạy (LẦN ĐẦU)

### Bước 1: Cài Python
- Tải tại: https://www.python.org/downloads/
- **QUAN TRỌNG**: Tick ✅ **"Add Python to PATH"** khi cài!

### Bước 2: Setup
- Double-click **`SETUP.bat`**
- Đợi nó cài xong (1-2 phút lần đầu)

### Bước 3: Chạy app
- Double-click **`START.vbs`** → App mở trong trình duyệt, không hiện terminal
- Hoặc **`ToolSpyIdea.bat`** → Có terminal (để debug nếu lỗi)

---

## 📦 Files

| File | Chức năng |
|------|-----------|
| `SETUP.bat` | Cài dependencies (chạy 1 lần) |
| `START.vbs` | Chạy app ẩn (double-click hàng ngày) |
| `ToolSpyIdea.bat` | Chạy app có terminal (debug) |
| `STOP.bat` | Tắt app đang chạy ngầm |
